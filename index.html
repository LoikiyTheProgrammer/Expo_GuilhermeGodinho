API vs Single Page Application (SPA)

API significa Interface de Programação de Aplicativos. É um conjunto de endpoints que permite que diferentes softwares se comuniquem entre si.

Uma Single Page Application (SPA) é um tipo de aplicativo web que carrega uma única página HTML e atualiza o conteúdo dinamicamente, sem recarregar a página inteira.

Exemplo de API:

      
        GET /api/users - Retorna a lista de usuários.
        POST /api/users - Cria um novo usuário.
        PUT /api/users/{id} - Atualiza os dados de um usuário existente.
        DELETE /api/users/{id} - Remove um usuário.

Exemplo de Single Page Application:

Uma SPA como o Facebook permite que você navegue entre diferentes seções (Feed de notícias, Perfil, Mensagens) sem recarregar toda a página.
